# Portfolio

## Compile & Deploy

npm run deploy

## Environment Setup

Create a `.env` file in the project root with:

* VITE_SERVICE_ID=your_service_id
* VITE_TEMPLATE_ID=your_template_id  
* VITE_PUBLIC_KEY=your_public_key  

Get these values from: https://www.emailjs.com
